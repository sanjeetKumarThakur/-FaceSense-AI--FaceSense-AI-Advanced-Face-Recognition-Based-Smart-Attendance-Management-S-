# Registration Form Application

## Overview
The Registration Form Application is a simple Tkinter-based GUI application that allows users to register by providing their personal information. The application features a visually appealing interface with background images and a structured layout for user input.

## Features
- User-friendly registration form
- Input fields for first name and other personal details
- Attractive design with background images

## Requirements
To run this application, you need to have the following installed:
- Python 3.x
- Tkinter (usually included with Python installations)
- Pillow (for image handling)

You can install Pillow using pip:
```
pip install Pillow
```

## Running the Application
1. Clone the repository or download the project files.
2. Navigate to the project directory.
3. Run the `register.py` file using Python:
   ```
   python register.py
   ```
