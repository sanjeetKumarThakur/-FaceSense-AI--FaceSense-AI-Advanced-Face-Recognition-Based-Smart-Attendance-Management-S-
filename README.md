# Login System with Tkinter

This project is a simple login system built using Python's Tkinter library for the GUI. It features:

- **Login Window**: Allows users to enter their username and password.
- **Registration**: (To be implemented) New users can register.
- **Password Reset**: (To be implemented) Users can reset forgotten passwords.
- **Image Integration**: Uses images for UI enhancement.

## Requirements

- Python 3.x
- tkinter
- Pillow (PIL)

Install dependencies with:
```
pip install pillow
```

## Usage

1. Place your background and icon images in the specified paths or update the code with your image paths.
2. Run the application:
   ```
   python login.py
   ```

## File Structure

- `login.py` - Main application file.
- `README.md` - Project documentation.

## Notes

- Registration and password reset features are placeholders and need backend/database integration.
- Update image paths in `login.py` as per your local setup.
